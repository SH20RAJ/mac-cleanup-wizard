Privacy Policy

Last updated: June 7, 2025

Information We Collect

Mac Cleanup Wizard is designed with privacy in mind. The application:

  • Does not collect personal data - We do not gather, store, or transmit any personal information
  • Works entirely offline - All scanning and cleaning operations happen locally on your Mac
  • No analytics or tracking - We do not use any analytics services or tracking mechanisms
  • No network connections - The app does not send data to external servers

File Access

Mac Cleanup Wizard requires access to your file system to:

  • Scan directories for junk files and caches
  • Calculate file and directory sizes
  • Move files to trash or delete them (only with your explicit consent)

All file operations are performed locally and no file contents are transmitted anywhere.

Permissions

The application may request the following macOS permissions:

  • Full Disk Access - To scan system directories and caches
  • File System Access - To analyze and clean files in your home directory

These permissions are used solely for the app's core functionality and are never used for data collection.

Data Security

Since Mac Cleanup Wizard operates entirely offline and does not collect or transmit data:

  • Your files and data remain on your device
  • No data is stored on external servers
  • No user accounts or registration required
  • No cookies or tracking technologies used
